# 🩺 AI Health Tracker - Dinda

Aplikasi pemantauan kesehatan interaktif berbasis Web untuk membantu pengguna melacak IMT (Indeks Massa Tubuh) dan tren berat badan secara real-time.

## 🚀 Fitur Utama
- **Automated BMI Calculator**: Menghitung skor IMT secara instan berdasarkan input pengguna.
- **Dynamic Data Visualization**: Menampilkan grafik tren kesehatan untuk pemantauan jangka panjang.
- **Health Status Classification**: Memberikan kategori kesehatan (Ideal, Overweight, dll) secara otomatis.

## 🛠️ Tech Stack
- **Python**: Bahasa pemrograman utama.
- **Streamlit**: Framework untuk membangun antarmuka web.
- **Pandas**: Digunakan untuk manajemen data (file .csv).

## 💻 Cara Menjalankan
1. Clone repo ini.
2. Pastikan sudah instal streamlit: `pip install streamlit`
3. Jalankan aplikasi: `streamlit run app_imt.py`
